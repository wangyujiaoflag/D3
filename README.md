- D3 数据可视化学习

  - D3（或 D3.js）是一个用于可视化数据的免费开源 JavaScript 库。其基于 web 标准的低级别方法在创作动态、数据驱动的图形方面提供了无与伦比的灵活性
  - D3 不是传统意义上的图表库。它没有“图表”的概念。当您使用 D3 可视化数据时，您可以组成各种基元
